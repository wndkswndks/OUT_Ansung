LoRa 집중 장치용 util_boot 프로그램
=============================================


1. 소개
----------------

이 소프트웨어는 PicoCell Gateway 부트로더로 점프하는 데 사용됩니다.
새 펌웨어로 MCU 플래시를 프로그래밍하십시오.


2. 사용
--------

사용 가능한 모든 옵션 목록을 보려면 명령줄 도움말을 참조하세요.
./util_boot -h
