LoRa 게이트웨이 프로젝트
======================

## 1. 핵심 라이브러리: libloragw
--------------------------

이 디렉토리에는 LoRa Picocell을 빌드하기 위한 라이브러리의 소스가 포함되어 있습니다.
Semtech LoRa 다중 채널 RF 수신기(집선기라고도 함)를 기반으로 하는 게이트웨이.
일단 컴파일되면 모든 코드는 libloragw.a 파일에 포함됩니다.
정적으로 링크됨(즉, 최종 실행 파일에 통합됨).
라이브러리는 집선기 임베디드 MCU와의 통신을 구현합니다.
직렬 포트를 통해.

라이브러리에는
라이브러리의 다른 하위 모듈을 테스트합니다.

## 2. 도우미 프로그램
----------------------

이러한 프로그램은 프로젝트에 포함되어 사용 방법에 대한 예를 제공합니다.
HAL 라이브러리와 시스템 빌더가 라이브러리의 다른 부분을 테스트하는 데 도움이 됩니다.

### 2.1. util_pkt_logger ###

이 소프트웨어는 JSON 구성을 사용하여 LoRa 집중 장치를 설정하는 데 사용됩니다.
파일을 생성한 다음 로그 파일에 수신된 모든 패킷을 무기한 기록합니다.
사용자가 응용 프로그램을 중지합니다.

### 2.2. util_com_stress ###

이 소프트웨어는 호스트 간의 링크의 신뢰성을 확인하는 데 사용됩니다.
플랫폼(프로그램이 실행되는 플랫폼) 및 LoRa 집중 장치 등록 파일
LoRa 집중 장치와의 모든 상호 작용을 통한 인터페이스입니다.
발생합니다.

### 2.3. util_tx_test ###

이 소프트웨어는 LoRa 집중 장치로 테스트 패킷을 보내는 데 사용됩니다. 패킷
프로토콜(예: MAC 주소) 정보에 대한 정보가 거의 포함되어 있지 않지만
다른 방법을 사용하여 게이트웨이 다운링크의 기능을 평가하는 데 사용할 수 있습니다.
게이트웨이를 수신기로 사용합니다.

### 2.4. util_tx_continuous ###

이 소프트웨어는 Tx 연속 모드에서 LoRa 집중 장치를 설정하는 데 사용되며,
스펙트럼 측정용.

### 2.5. util_boot ###

이 소프트웨어는 프로그래밍을 위해 PicoCell Gateway 부트로더로 점프하는 데 사용됩니다.
새로운 펌웨어로 MCU.
에 대한 자세한 내용은 picoGW_mcu 저장소의 readme를 참조하십시오.
MCU 플래시 프로그래밍.

### 2.6. util_chip_id ###

이 소프트웨어는 PicoCell 게이트웨이의 고유 ID(
STM32 고유 ID 레지스터에서 추출된 64비트 고유 ID).
